RU

1. Открываем файл "script_get_mp3_from_video_file.py" через любую IDE.

2. Добавляем в папку, где лежит "script_get_mp3_from_video_file.py" видеофайл.

3. Изменяем в строчке: video_file = Path('my_video.mp4') на имя вашего видеофайла, например: video_file = Path('Видео.mp4').

4. Запускаем скрипт. В папке, где лежит файл "script_get_mp3_from_video_file.py" появится ".mp3" файл с извлеченной из видео аудиодорожкой.

EN

1. Open the file "script_get_mp3_from_video_file.py" using any IDE.

2. Add a video file to the folder where "script_get_mp3_from_video_file.py" is located.

3. Modify the line: `video_file = Path('my_video.mp4')` to the name of your video file, for example: `video_file = Path('Video.mp4')`.

4. Run the script. A ".mp3" file with the extracted audio track from the video will appear in the folder where the "script_get_mp3_from_video_file.py" file is located.
